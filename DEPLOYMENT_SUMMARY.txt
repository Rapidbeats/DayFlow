┌──────────────────────────────────────────────────────────────────────────┐
│                    📦 WHAT TO UPLOAD TO CLOUDFLARE                       │
└──────────────────────────────────────────────────────────────────────────┘

✅ UPLOAD THESE (from dist/ folder):

    dist/
    ├── index.html           ← Main HTML file
    ├── assets/              ← JavaScript + CSS
    │   ├── index-xxx.js
    │   ├── index-xxx.css
    │   └── ...
    └── _redirects           ← Routing config

❌ NEVER UPLOAD THESE:

    node_modules/            ← Too large (500+ MB)
    src/                     ← Source code (not needed)
    package.json             ← Config (not needed)
    vite.config.ts           ← Config (not needed)
    *.ts, *.tsx files        ← Source files (not needed)

┌──────────────────────────────────────────────────────────────────────────┐
│                      ⚡ COMMON ISSUES & FIXES                            │
└──────────────────────────────────────────────────────────────────────────┘

Problem: "pnpm: command not found"
Fix: npm install -g pnpm

Problem: "dist folder doesn't exist"
Fix: pnpm run build

Problem: "404 errors on deployed site"
Fix: Make sure _redirects file is in dist/

Problem: "Can't log in on deployed site"
Fix: Check Supabase credentials in src/app/lib/supabase.ts

Problem: "Tasks don't sync"
Fix: Verify Supabase database setup (run SQL migration)

For more solutions → Read TROUBLESHOOTING.md
